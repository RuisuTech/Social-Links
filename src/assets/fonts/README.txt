## Inter Variable Font

Este paquete de descarga contiene Inter tanto como una fuente variable como fuentes estáticas.

**Inter** es una fuente variable con los siguientes ejes:
- `slnt` (inclinación)
- `wght` (peso)

Esto significa que todos los estilos están contenidos en un solo archivo:
- `Inter-VariableFont_slnt,wght.ttf`

Si tu aplicación soporta completamente fuentes variables, ahora puedes seleccionar estilos intermedios que no están disponibles como fuentes estáticas. No todas las aplicaciones soportan fuentes variables, y en esos casos puedes usar los archivos de fuentes estáticas para Inter:

- `static/Inter-Thin.ttf`
- `static/Inter-ExtraLight.ttf`
- `static/Inter-Light.ttf`
- `static/Inter-Regular.ttf`
- `static/Inter-Medium.ttf`
- `static/Inter-SemiBold.ttf`
- `static/Inter-Bold.ttf`
- `static/Inter-ExtraBold.ttf`
- `static/Inter-Black.ttf`

### Para empezar

1. Instala los archivos de fuente que deseas usar.
2. Utiliza el selector de fuentes de tu aplicación para ver la familia de fuentes y todos los estilos disponibles.
